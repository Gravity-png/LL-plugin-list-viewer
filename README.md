# LiteLoaderQQNT Plugin 插件列表查看

> [!NOTE]
> 该插件仅在 Windows 环境下测试开发，未对其他平台进行测试，不保证可用性

## 功能

- 插件列表查看
- 插件检查更新
- 插件安装（支持镜像）
- 插件卸载
- 插件查找

![](./imgs/1.png)

## 使用方法

### 下载发行版

1. 下载发行版并解压
2. 将文件夹移动至 `LiteLoaderQQNT数据目录/plugins/` 下面
3. 重启 QQNT
